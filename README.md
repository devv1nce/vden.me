
# vden.me — Game Developer Portfolio

Personal portfolio website built with React, showcasing my game development projects, skills, and experience.

🌐 **Live site:** [vden.me](https://vden.me)

## Features

- 🎮 Games portfolio with personal & school project tabs
- ▶️ Play games directly in the browser or link to external platforms
- ⚙️ In-development project status badges
- 📄 School project document viewer (PDF / Google Docs)
- 💻 Interactive code editor hero section
- 📊 Skills section with progress bars
- 📬 Contact form
- 📱 Fully responsive

## Tech Stack

- React.js
- CSS3 (custom properties, animations, clip-path)
- react-scroll
- react-slick

## Fonts

- Orbitron — headings
- Rajdhani — body
- Share Tech Mono — code / labels
